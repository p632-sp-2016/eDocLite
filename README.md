# eDocLite

Steps to Run: 

1) Install Dependencies
      npm install

2) Run Application
      npm start
      

Steps to Test: 

1) Start Karma Server
      npm run test
      
2) Open http://localhost:9876 in Chrome


Steps to generate documents using esdoc: 

1) npm run docs

Steps to for using eslint:

1) npm run lint
